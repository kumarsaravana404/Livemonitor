HOW IT WORKS

Integrate in minutes, protect forever

1

CREATE AN ACCOUNT & GET YOUR API KEY

Register for free, add the email you want to protect, and get a unique API key instantly.

2

ADD ONE LINE TO YOUR LOGIN FORM

Call our API endpoint whenever a login attempt happens on your platform. Works with any backend.

POST https://yoursite.com/api/report-attempt Authorization: Bearer sw_your_api_key { "account_id": "your-monitored-account-id", "ip_address": "captured-from-request", "user_agent": "browser-user-agent", "status": "FAILED" }
3

MONITOR YOUR DASHBOARD IN REAL TIME

See every attempt live — with IP address, location, ISP, device info, and attempt counter.

4

GENERATE REPORTS & FILE COMPLAINTS

Click any attempt to generate a full forensic report. Use it to file a complaint at cybercrime.gov.in or your country's cyber crime authority. Law enforcement can then legally compel the ISP to reveal the attacker's identity.
